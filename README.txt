Task time tracker API - это проект, который может не только записывать задачи в БД, но и отслеживать время, затраченное на задачи. Функционал представляет собой создание, получение, изменение информации об задачах и времени. Работоспособность проверяется с помощью программы Postman по определенным end-поинтам.

Чтобы собрать проект, надо клонировать репозиторий - https://github.com/Petr0005/CDEK_task.git. Установить программу Docker, иметь версию Java 21, сборщик проекта Maven, запустить docker compose с помощью программы - docker-compose up -d

Программа проверяется через JUnit тесты и программу Postman. Вот ссылка на запросы
